五、知识总结

1、核心流程

登录系统 → 文件目录操作 → 用户与组管理 → 文件权限设置 → 环境变量查看与配置（临时/永久）

2、核心知识点

知识点 关键内容 命令 / 配置
系统信息查看 查看当前用户、内核版本、Ubuntu版本、主机名、当前路径 whoami
uname -a
lsb_release -a
hostname
pwd
文件目录基本操作 列出文件、切换目录、创建/复制/移动/删除文件 ls -lh
cd ~
mkdir, touch, cp, mv, rm -f
系统资源查看 内存、磁盘、进程、目录大小 free -h
df -h
top
du -sh *
用户与组配置文件 查看用户账户、加密密码、组信息、当前用户ID cat /etc/passwd
sudo cat /etc/shadow
cat /etc/group
id
用户管理 创建用户、设置密码、赋予sudo权限、查看组、切换用户、删除用户 sudo adduser testuser
sudo passwd testuser
sudo usermod -aG sudo testuser
groups testuser
su - testuser
sudo deluser -r testuser
文件权限管理 权限数字对照（7=rwx,6=rw-,5=r-x,4=r–,0=—）；数字法与符号法修改权限 chmod 755 file
chmod u+x file
chmod g-w file
chown user:group file
chgrp group file
环境变量查看 查看所有环境变量、PATH、HOME、USER env
echo $PATH
echo $HOME
echo $USER
临时设置环境变量 临时添加PATH、创建变量、验证、删除 export PATH=$PATH:/my/bin
export MY_VAR="Hello"
echo $MY_VAR
unset MY_VAR
永久设置（用户级） 修改 ~/.bashrc、~/.profile，添加环境变量或别名，保存后 source 生效 nano ~/.bashrc
末尾添加 export PATH=... 或 alias ll='ls -alF'
source ~/.bashrc
永久设置（系统级） /etc/environment（重启生效，不加export）
/etc/profile（所有用户登录执行）
/etc/profile.d/（推荐，单独脚本）		 sudo nano /etc/environment
sudo nano /etc/profile
sudo nano /etc/profile.d/myenv.sh 添加 export MY_VAR=...
source /etc/profile.d/myenv.sh
